## CVE's examples
We have collected some CVE's to give insight into what the CVS look like. The folder `GitHub`contains CVE's from GitHub, `NVD` is CVE's from NIST and the folder `RedHat` contains some CVE's from RedHat

Read more about CVE at [www.cve.org](https://www.cve.org/)

## KrafCERT
KraftCERT is a sector CERT for the electric power and petroleum sectors. KraftCERT is part of the Norwegian national response organization. KraftCERT update our members about relevant vulnerabilities and threats, so that they will be able to detect and defer digital attacks. All KBO units are obliged to send notifications to KraftCERT. Read more at [kraftCERT](https://www.kraftcert.no/no/) home page.

The REPO [kraftcert-varsler](https://github.com/krafthack-by-techday/kraftcert-varsler) contains examples of a typical vurnability warnings.
